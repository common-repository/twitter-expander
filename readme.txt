== Description ==

Expand your Twitter-Name @something in your Posting with Twitterlink http://twitter.com/something 

Manual:

If you want to link a Twittername write @twittername in your post. this Plugin will it automatically link

This Plugin is GDPR & DSGVO secure


== Installation ==

  1. Upload des Plugins in den Plugin-Ordner /wp-content/plugins/
  2. In der Plugin-Administration ihres Blogs kann nun Twitter Expander aktiviert werden.
  3. Wenn man nun bei einer Seite oder einem Beitrag den Twitter Name eingibt, dann wird dieser automatisch mit dem Twitterprofil verlinkt und man muss nicht mehr den Link von hand setzen
